# Robotic pipetter based on delta-robot

The delta robot is the repurposed Anycubic Kossel 3D printer. Controlled by streaming G-code
via the virtual serial port to the Arduino controlling the delta robot with standard
Kossel firmware.

`APT.dll` is the driver for Tholabs translational stage MTS50/M. 
This is the "APT" driver, discontinued by Thorlabs.
