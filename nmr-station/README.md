# Automated NMR Station

## Setup and Installation

### On the Working PC

Use python 3.10

Install required package
```
pip install -r requirements.txt
```

Run the application
```
python app.py
```

### Open the software for Spinsolve

Enable remote control

## How to Use

1. Access the web interface at `http://localhost:5000`
2. Fill in information for the testing sample
3. Select a protocol from the dropdown, input or select the parameters
4. Add the protocol to the list
5. Click 'Save' to update your expected method
6. Mark the grid that will perform analysis
7. Click 'Start' to begin the analysis
